# Website bavarianrankengine.com — Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Build a static HTML/CSS/JS one-pager marketing website for the Bavarian Rank Engine WordPress plugin, plus changelog and impressum subpages.

**Architecture:** Three plain HTML files sharing one CSS file and one JS file. No build step, no framework, no dependencies. Self-hosted fonts (DSGVO). Folder lives at `/var/www/dev/plugins/bre-dev/website/`.

**Tech Stack:** HTML5, CSS3 (custom properties, grid, flexbox), vanilla JS ES6+, woff2 self-hosted fonts (Playfair Display, Inter, JetBrains Mono).

**Design reference:** `docs/plans/2026-02-21-website-design.md`

(function () {
    'use strict';

    // ── Nav scroll behaviour ──────────────────────────────────
    const nav = document.getElementById('nav');
    if (nav) {
        const onScroll = () => {
            nav.classList.toggle('scrolled', window.scrollY > 50);
        };
        window.addEventListener('scroll', onScroll, { passive: true });
        onScroll();
    }

    // ── Mobile nav toggle ─────────────────────────────────────
    const toggle   = document.getElementById('navToggle');
    const navLinks = document.getElementById('navLinks');
    if (toggle && navLinks) {
        toggle.addEventListener('click', () => {
            navLinks.classList.toggle('open');
            toggle.setAttribute(
                'aria-expanded',
                navLinks.classList.contains('open') ? 'true' : 'false'
            );
        });

        // Close on link click
        navLinks.querySelectorAll('a').forEach(link => {
            link.addEventListener('click', () => {
                navLinks.classList.remove('open');
                toggle.setAttribute('aria-expanded', 'false');
            });
        });
    }

})();
